# 🧩 Next.js SaaS Starter Template

A modern and extensible Next.js 15 template designed for building full-stack SaaS products — fast.

## 🔧 Tech Stack

- ⚛️ **Framework:** Next.js 15 (App Router)
- 🎨 **UI:** Tailwind CSS + Shadcn UI
- 🔐 **Authentication:** Clerk
- 🧮 **Database:** Supabase (PostgreSQL)
- 💳 **Payments:** Stripe
- 📈 **SEO:** Optimized meta tags included
- 💡 **TypeScript:** Type-safe from end to end
